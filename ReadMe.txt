Machine Learning Model Evaluation Practical 📊

📌 Introduction

This practical is based on evaluating the performance of a machine learning classification model. It helps in understanding how well a model is predicting the correct output.

🎯 Objective

The main objective of this practical is to analyze the performance of a binary classification model using different evaluation metrics.

🛠️ Technologies Used
Python
NumPy
Matplotlib
Scikit-learn
⚙️ Working
First, actual values (y_true) and predicted probabilities (y_prob) are taken
Probabilities are converted into binary predictions (0 or 1)
Then different evaluation metrics are calculated
📊 Evaluation Metrics Used
Confusion Matrix
Accuracy
True Positive Rate (TPR)
False Positive Rate (FPR)
ROC Curve
📈 Output
The confusion matrix shows correct and incorrect predictions
Accuracy gives overall performance of the model
ROC Curve is plotted to visualize model performance
📚 Learning Outcome
Understanding of classification models
How to evaluate model performance
Basics of ROC Curve and confusion matrix
